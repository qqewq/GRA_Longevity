# GRA Longevity: 150+ лет здоровой жизни через многоуровневую GRA-обнулёнку

Проект объединяет биологию, геронтологию и аппарат Geometric Recursive Analytics (GRA) для системного подавления «пены старения».  
Платформа реализует **in silico эксперименты** на виртуальных пациентах с верификацией на известных геропротекторах.

## Основные концепции
- **Пена старения Φ**: отклонение ключевых биомаркеров от оптимального гомеостаза.
- **Многоуровневый мультиверс**: от эпигенетических часов до органных систем, согласованных рекурсивными проекторами.
- **Все виды GRA**: статическая (базовый гомеостаз), циклическая (циркадные ритмы), хаотическая (здоровый хаос ВСР, иммунитет), гибридная.
- **In silico эксперименты**: виртуальная когорта, обучение GRA-контроллера максимизировать lifespan (lifetime reward).
- **Верификация**: сравнение с эффектами рапамицина, метформина, холодового воздействия и др.

## Структура проекта
- `src/` – движок GRA, модель пациента, RL-оптимизатор
- `config/` – конфигурации типов обнуления для разных систем
- `data/` – синтетические/реальные траектории биомаркеров
- `experiments/` – скрипты запуска и анализа
- `notebooks/` – демонстрации
